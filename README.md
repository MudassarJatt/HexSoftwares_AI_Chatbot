# 🤖 AI Chatbot Project

This is a simple AI Chatbot developed using Python and Natural Language Processing (NLP).

## 📌 Features
- Responds to user input
- Basic NLP using NLTK
- Interactive conversation

## 🛠️ Technologies Used
- Python
- NLTK

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run chatbot:
   python chatbot.py

## 🎯 Project Objective
To build a simple AI chatbot that can interact with users in human-like conversation.
